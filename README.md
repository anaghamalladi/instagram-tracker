# Instagram Activity Tracker

A web app to track time spent on specific Instagram activities — Reels, Messages, Posts, and Stories — separately.

## Live Demo
[View App](https://anaghamalladi.github.io/instagram-tracker/)

## Features
- Per-activity timer (Reels, Messages, Posts, Stories)
- Daily summary with bar chart
- 7-day history with stacked bar chart
- Custom daily time limits with alerts
- Reset button to clear today's data
- Data persists across sessions using localStorage

## Tech Stack
- HTML, CSS, JavaScript
- Chart.js for data visualization
- localStorage for data persistence

## How to Use
1. Set your daily time limits for each activity
2. Hit Start when you open Instagram and switch to an activity
3. Hit Stop when you switch away
4. Click Show Summary to see today's breakdown and chart
5. Click 7-Day History to see your usage trend
